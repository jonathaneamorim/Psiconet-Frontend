# FormLogin
Componente utilizado para centralizar os métodos de login, podendo ser chamado em qualquer módulo apenas recenbendo um parâmetro chamado loginType que vai identificar qual tipo de login está sendo realizado.

Esse componente é integrado com o [Action (Auth.ts)](../actions/Auth.md) que realiza todo o esquema de autenticação e armazenamento do token de usuário.
